# Репетитор по немецкому A0–B2

Публичное HTML-приложение для самостоятельного изучения немецкого языка от полного новичка до B2.

## Быстрые входы

- **Я новичок** — старт с A0.
- **Проверить уровень** — диагностика.
- **Грамматика B2 коротко** — отдельный вход в главные темы B2.
- **Словарь отдельно** — фильтры по уровню, теме и поиску.

## Состав v14.1

- уровни A0 / A1 / A2 / B1 / B2;
- уроки: **78**;
- словарь и фразы: **464**;
- диагностические вопросы: **31**;
- экзаменационные промпты: **7**;
- прогресс через `localStorage`;
- кнопка `✓ изучено`;
- отдельный режим «скрывать изученное»;
- профильные темы: медицина, клиника, Швейцария, пластическая хирургия;
- мобильная навигация;
- печатный режим.

## Проверка

Локальная визуальная проверка выполнена через Chromium/Playwright с `page.set_content`, потому что в этой среде прямые переходы `file://` и `localhost` блокировались политикой окружения. Сохранены скриншоты desktop/mobile в `docs/screenshots/` и метрики в `data/visual_check_metrics_v14_1.json`.
